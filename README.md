角色信息提取 Chrome 插件
一款高效提取页面角色信息及关联价格的 Chrome 插件，专为解析特定结构页面设计。
✨ 功能特性
自动识别页面中 ID 以 other_info_ 开头的文本区域
精准解析文本区域内的 JSON 格式角色数据
智能提取角色名称（优先从页面元素获取， fallback 至 JSON 数据）
自动关联并格式化角色对应价格（基于 data_price 属性）
结构化输出所有提取数据，支持批量查看
📦 安装步骤
由于插件暂未发布至 Chrome 应用商店，需通过开发者模式手动安装：
获取代码
点击右上角 Code 按钮 → 选择 Download ZIP
解压至本地任意目录（如 ./character-extract-plugin）
开启开发者模式
打开 Chrome 浏览器，访问 chrome://extensions/
开启右上角 开发者模式 开关（开启后将显示额外操作按钮）
加载插件
点击左上角 加载已解压的扩展程序
选择之前解压的插件目录
加载成功后，插件将显示在扩展列表中
🚀 使用方法
打开包含角色信息的目标页面
点击浏览器工具栏中的插件图标（若未显示，可点击扩展程序按钮后选择本插件）
插件将自动开始提取数据
提取完成后，在弹窗中查看结构化结果
⚠️ 注意事项
仅支持特定结构页面，页面布局变更可能导致提取失败
依赖页面中存在 ID 以 other_info_ 开头的 textarea 元素
价格提取依赖 data_price 属性，无此属性时显示 "未知价格"
功能异常时，可尝试在扩展页面点击插件的 重新加载 按钮
❓ 常见问题
Q: 安装后插件无反应？
A: 请检查：
是否开启开发者模式
插件目录是否正确加载
目标页面是否符合解析要求
Q: 提取的数据不完整？
A: 可能是部分 textarea 中的 JSON 格式有误，可按 F12 打开控制台查看具体错误信息。
Q: 如何更新插件？
A: 重新下载最新代码替换原有目录，在扩展页面点击 重新加载 即可。

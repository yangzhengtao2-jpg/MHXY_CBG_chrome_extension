角色信息提取 Chrome 插件
一款用于提取页面中角色信息的 Chrome 浏览器插件，可自动解析特定结构页面中的角色数据及关联价格信息。
功能说明
自动识别页面中 ID 以 other_info_ 开头的文本区域（textarea）
解析文本区域中的 JSON 格式角色数据
从页面元素中提取角色名称和对应价格信息
整合所有提取的数据并提供结构化输出
安装步骤
由于本插件暂未发布到 Chrome 应用商店，需通过开发者模式手动安装，步骤如下：
下载插件代码
点击页面右上角的 Code 按钮，选择 Download ZIP 下载插件压缩包
解压下载的 ZIP 文件到本地任意目录（例如 ./chrome-character-extractor）
开启 Chrome 开发者模式
打开 Chrome 浏览器，在地址栏输入 chrome://extensions/ 进入扩展程序页面
打开页面右上角的 开发者模式 开关（开启后会显示额外的操作按钮）
加载插件
点击页面左上角的 加载已解压的扩展程序 按钮
在弹出的文件选择框中，选择之前解压好的插件目录
成功加载后，插件会显示在扩展程序列表中
使用方法
打开包含角色信息的目标页面
点击浏览器工具栏中的插件图标（如果未显示，可点击扩展程序图标后选择本插件）
插件会自动提取页面中的角色信息
提取完成后，可在插件弹窗中查看或导出数据
注意事项
本插件仅适用于特定结构的页面，若页面结构发生变化可能导致提取失败
确保目标页面中存在 ID 以 other_info_ 开头的 textarea 元素
价格信息提取依赖页面中特定的 data_price 属性，若页面无此属性将显示 "未知价格"
安装后若功能异常，可尝试在扩展程序页面点击插件的 重新加载 按钮
常见问题
Q: 安装后插件无法正常工作？
A: 请检查是否开启了开发者模式，插件目录是否正确加载，以及目标页面是否符合解析要求。
Q: 提取的数据不完整？
A: 可能是部分 textarea 中的 JSON 格式有误，可打开浏览器控制台（F12）查看具体错误信息。
Q: 如何更新插件？
A: 重新下载最新代码并替换原有目录，然后在扩展程序页面点击 重新加载 即可。
